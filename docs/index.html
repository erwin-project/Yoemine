Manage your Mining Coal Production using Yeomine

Comprehensive issues in open pit coal mining include overburden removal and coal recovery issues. Coal production is strongly influenced by productivity. Based on the data in mining company, there were 8% of the coal production target that was not achieved or equivalent to 3.3 billion rupiah. So, that's why Yeomine present to solve that problem.

About Yeomine

Yeomine is a product that is built by a web and desktop application using python language as backend and streamlit as framework. This is integrated with computer vision technology using YoloV8 Model that is developed with thousands of actual and valid open coal mining data.

In Yeomine, there is a recommendation and prediction system that is very helpful in achieving coal extraction targets in open pit mines. This system or application can help maintain energy stability by optimizing the control of coal production.

Object Detection

There are many object detections provided by Yoemine.

Coal Getting

Can detect the several kind of coal stone like clean coal, non-clean coal, clean spoil, spoil, digging limit, measurable roof, water

Typical Seam

Can detect the several kind of seam gb such as: seam 5, seam 5a1, seam 5a2, seam 5a2u, seam 5a3, seam 5a4, seam 5b, undefined seam.

Core Logging

Can detect the several kind of core logging such as: bedrock ultramafik, limonit, saprolit. And also, can detect the length of core logging.

HSE Safety

Can detect people/staff who use and don't use standard clothes for HSE safety. For the example like helmet, glass eye, etc.

General Custom

Can detect specific object in the mining field based on that you want. For the example truck, structure geology, etc.

*Fuel Scheduling

Gives recommendation about when the coal hauling and fuel refuel based on efficiency time and budget.

Features App

The several features that provided by Yeomine.

Smart Detection

Can detect objects in the form of videos, images, or automated webcams. As for the objects handled by Yeomine include coal, core-logging, seam, HSE attribute, and general case in mining field.

Build custom model

Yeomine provides a training model that can later be trained according to the needs of users in their respective companies. The trained model can use data from the admin or from the user.

Cloud base computation

All data processing up to the creation of machine learning models are carried out in the cloud base. This makes it easier for users not to need to set up a high computing PC.

Real time data

The model AI that has been created will be updated along with data adding data every day. This is to maintain the quality of the object detection results in the future.

Direct download file

All predictable data or models that have been trained, can be downloaded directly. This can help the user in analyzing the quality of the predicted object.

User interface flexibility

Users can access the Yeomine application both on the website and desktop for PCs or laptops. Even users can access the Yeomine webs application via a smartphone.
